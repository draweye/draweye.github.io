Hyper-V ​

Hyper-V 建议安装Windows系统，Linux的话强烈推荐使用WSL。
安装就不多说了，启用或关闭Windows功能里直接勾选就可以，更高级的需求看下官方文档即可。
如果跑大模型相关的软件，默认安装的虚拟机是不支持GPU运算的，所以这里主要介绍下如何使用Hyper-V虚拟化显卡。

WARNING

本文档针对Nvidia 显卡，其他显卡需要自己找教程。

开始前的准备 ​

  • 关闭虚拟机
  • 下载脚本包，然后解压到D:\Shell目录。
    如果虚拟机还未安装系统，这个包里可以在没有装虚拟机的情况下直接安装，具体看参考资料中github的说明。以下是针对已经安装好系统的情况。
  • 在D:\Shell中新建一个 setvm.ps1 文件内容如下
$vm = "VM_NAME"
if (Get-VMGpuPartitionAdapter -VMName $vm -ErrorAction SilentlyContinue) {
   Remove-VMGpuPartitionAdapter -VMName $vm
}
Set-VM -GuestControlledCacheTypes $true -VMName $vm
Set-VM -LowMemoryMappedIoSpace 1Gb -VMName $vm
Set-VM -HighMemoryMappedIoSpace 32Gb -VMName $vm
Add-VMGpuPartitionAdapter -VMName $vm

开始 ​

在管理员模式下打开 Windows PowerShell

进入刚解压的目录

PowerShell
cd D:\Shell
1

检查权限

PowerShell
Get-ExecutionPolicy
1

如果输出的是Restricted则执行

PowerShell
Set-ExecutionPolicy Unrestricted
1

执行脚本

PowerShell
.\Update-VMGpuPartitionDriver.ps1 -VMName "VM_NAME" -GPUName "AUTO"
.\setvm.ps1
