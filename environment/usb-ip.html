USB/IP ​

把USB设备通过网络在其他机器上加载。
适用场景例如RPA自动化时网银U盾共享、Hyper-V虚拟机中加载、WSL中加载。

服务端 ​

下载安装包 或者直接用命令行安装

PowerShell
winget install usbipd
1

使用说明 ​

管理员权限运行PowerShell

查看设备

PowerShell
usbipd list
1

分享设备

PowerShell
usbipd bind --busid=BUSID
1

TIP

将BUSID替换为刚才查看设备时要分享的设备的BUSID

取消分享

PowerShell
usbipd unbind --busid=BUSID
1

TIP

将BUSID替换为刚才查看设备时要分享的设备的BUSID

客户端 ​

安装前准备

管理员权限运行PowerShell

PowerShell
bcdedit.exe /set testsigning on
1

TIP

成功后重启计算机

如果输出异常

PS C:\Windows\System32> bcdedit.exe /set TESTSIGNING ON
设置元素数据时出错。
该值受安全引导策略保护，无法进行修改或删除。
PS C:\Windows\System32>
1
2
3
4

是因为启用了安全启动（Secure Boot），这使得某些与安全启动策略冲突的设置无法修改。在这种情况下，您需要暂时禁用安全启动，以便能够修改测试签名（Test Signing）设置。

Hyper-V虚拟机的话修改安全设置，把“启用安全启动”前面的勾去掉（先关闭虚拟机）

物理机请修改BIOS中相关设置

安装配置

下载安装包，安装后打开软件。

填入Server的IP地址，点Add Device按钮
选中要连接的设备后，点Attach按钮

正常连接后，客户端就可以看到USB设备了
